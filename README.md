# MCLazyDatapack

> Cubik 自制 Minecraft 懒人合成数据包

------

~~**NOT VANILLA !!!**~~

功能列表：

- [X] 2线合成为1白地毯
- [X] 1白地毯分解成2线
- [X] 1白羊毛分解成4线（原版提供4线合成1羊毛）
- [X] 2地毯合成为1羊毛（颜色不变）
- [X] 使用以下两种方法合成海绵

    ![sponge_with_yellow_wool](sponge_with_yellow_wool.png)
    ![sponge_with_hay_block](sponge_with_hay_block.png)
